# book-tracker

## Автор
София Арчакова

## Описание
приложение для отслеживания прочитанных книг. позволяет добавлять книги, фильтровать их и сохранять данные в JSON.

## Возможности
- добавление книги
- фильтр по жанру
- фильтр по количеству страниц
- сохранение данных

## Запуск
1. установите Python
2. запустите : python main.py

## Пример использования
- введите книгу
- нажмите "Добавить книгу"
- используйте фильтры
